== Description ==

Mail Picker is latest addition for email marketing and sending newsletter made easy, build unlimited subscriber list and send unlimited email to your subscriber.


### 3rd Party Plugins Support


**Contact form 7**
Contact form 7 is one the most popular form builder plugin for WordPress, so we added functionality to grab email and
name on form submission. you can choose email list to assign them and send confirmation email and welcome email also.

**WooCommerce**
WooCommerce is the best eCommerce plugin for WordPress, so we added functionality to grab email and user details for
building email list from customer orders, you can grab customer email, phone, first name,last name, country and etc, you
 can also send them confirmation email and welcome email too.

**Contact form by WPForms**
WPForms is another most popular form builder plugin and we added in our plugin to grab user form submission to get email
 and name to build your email list. also you can send them confirmation email and welcome email too.

**Flamingo**
Flamingo is popular for saving contact form 7 submission so we added in our plugin to grab user submission to get email
and name, you can also send them confirmation email and welcome email.


**Ninja Forms**
Ninja Forms is another best form builder plugin, it has built-in form submission features, so we added functionality to
grab email and user name to build your own email subscriber list. you can also send conformation and welcome email to
each subscriber.

**Everest Forms**
Everest Forms also another popular form builder plugin and come with built-in form submission feature, so we added
functionality to our plugin to get user email and name to build email subscriber list.

**Forminator forms**
Forminator is another cool plugin to build contact form and payment forms, we added functionality to grab email and user
 name and add them to email list.

**More 3rd party plugins**

* weForms
* Caldera Forms
* Kali Forms



### Features

**Unlimited Subscriber List**
You can create unlimited number of subscriber and list, we never limit you to go beyond.

**Customizable Opt-in Forms**
There is ready form builder to build opt-in form, you can add/remove any input field as you want, you can also add custom input field and save info to custom meta fields.

**Mail Templates**
You can create your own templates by using any page builder like Gutenberg and Elementor and any other page builder, there is some ready tags you can make email template dynamic.
